# Sistema de Gestión de Inventario

## Descripción
Sistema web desarrollado con Spring Boot y PostgreSQL para gestionar el inventario de productos en múltiples sucursales. Permite registrar movimientos de entrada, salida y ajuste, controlar stock por sucursal, generar alertas automáticas y exportar reportes.

## Tecnologías
- Java 21
- Spring Boot 3
- Spring Data JPA / Hibernate
- PostgreSQL
- Thymeleaf
- Maven

## Funcionalidades
- CRUD de Productos
- CRUD de Categorías
- CRUD de Sucursales
- Registro de movimientos de inventario (Entrada, Salida, Ajuste)
- Control de stock por sucursal
- Alertas automáticas de stock mínimo
- Reportes filtrados por producto, sucursal y fecha
- Exportación de reportes a CSV
- Paginación con Spring Pageable
